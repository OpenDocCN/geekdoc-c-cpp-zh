# 输出参数

> 原文：[`cel.cs.brown.edu/crp/idioms/out_params.html`](https://cel.cs.brown.edu/crp/idioms/out_params.html)

C++ 中有几个惯用方法涉及使用 out 参数：将指针或引用传递给函数，以便函数可以修改它们以提供其结果。

本节中的章节讨论了在 Rust 中实现与 C++ 中使用 out 参数相同目标的惯用方法。许多 Rust 惯用方法类似于针对较新 C++ 标准编程时推荐的替代 out 参数的方法。
