# C++到 Rust 短语手册

> 原文：[`cel.cs.brown.edu/crp/title-page.html`](https://cel.cs.brown.edu/crp/title-page.html)

本书旨在帮助 C++程序员学习 Rust。它将常见的 C++模式翻译成惯用的 Rust。每个模式都通过具体的代码示例以及高级的工程权衡讨论来描述。

本书可以从前到后阅读，但它设计成可以随机访问。当你编写 Rust 代码并想，“我知道如何在 C++中这样做，但不知道如何在 Rust 中做”时，请查找本书中相应的章节。

这本书是由布朗大学认知工程实验室的专家 C++和 Rust 程序员手写的。[认知工程实验室](https://cel.cs.brown.edu/)的目标是提供准确的信息，并带有适当的细节程度。本书中的所有文本均非 AI 撰写。

## 其他资源

如果你没有任何 Rust 经验，你可能首先考虑阅读[《Rust 编程语言》](https://rust-book.cs.brown.edu/)或快速浏览[Learn X in Y Minutes](https://learnxinyminutes.com/rust/)。

如果你主要使用 C 或 C++进行嵌入式系统编程，这本书是[《嵌入式 Rust 手册》](https://docs.rust-embedded.org/book/)的补充。

与像[Rustonomicon](https://doc.rust-lang.org/nomicon/)和[用大量链表学习 Rust](https://rust-unofficial.github.io/too-many-lists/)这样的资源相比，本书更少关注“Rust 幕后”，而是明确描述了 Rust 在 C++术语下的工作方式。
